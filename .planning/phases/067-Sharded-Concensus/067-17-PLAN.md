# 067-17 Structured Evidence Registry

<objective>

digest-bound records so the simulator, validators, watchers, and final report
can audit equivocation, payload withholding, missing blob, wrong root, wrong
route digest, stale member, and split-brain events without relying on log text.
</objective>
