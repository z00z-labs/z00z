# 067-02 Commit Subject And Certificate Types

<objective>
Promote the local quorum decision from `JournalCandidate` plus voter id list to
first-class protocol artifacts with canonical binary encoding, stable digests,
real membership semantics, and fail-closed vote/certificate validation.
</objective>
