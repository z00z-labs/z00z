## 이 화면 사용 {#current-view}
- 전체, 코인, 토큰, NFT로 선택한 지갑의 자산 목록을 좁힙니다.
- 잔액은 지갑 소유 데이터입니다. 신뢰할 수 있는 시세 피드가 없으면 가치와 가격은 사용할 수 없음으로 남습니다.

## 로컬 및 안전 동작
- 행을 선택하면 읽기 전용 자산 메타데이터가 열리며 보내기와 받기는 별도의 지갑 작업입니다.
- 자산 아이콘과 도움말은 앱에 포함되어 오프라인에서도 작동합니다.
