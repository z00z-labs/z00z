## 이 화면 사용 {#current-view}
- 먼저 자산, 바우처 또는 권한을 선택합니다. 각각 가치, 정책 기반 조건부 가치, 가치가 0인 제한된 권한입니다.
- 한 번 승인하기 전에 수신자와 잔액 또는 정책, 만료, 남은 사용 횟수, 범위, 위임 조건을 확인합니다.

## 로컬 및 안전 동작
- 지갑 비밀과 개인 전송 데이터는 도움말에 포함되지 않습니다.
- 도움말은 앱에 포함되어 오프라인에서도 작동합니다.
